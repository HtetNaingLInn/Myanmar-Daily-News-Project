# Myanmar Daily News Project

### Installation

```
-git clone https://github.com/HtetNaingLInn/News-media--project.git PROJECT_FOLDER_NAME
-cd PROJECT_FOLDER_NAME
-composer install
-cp .env.example .env
-Config database
-php artisan key:generate
-Done.

```
